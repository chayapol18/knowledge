# <HTML> โครงสร้างหน้าเว็บ

## HyperText Markup Language

### ประกอบไปด้วย 
    - Tags <>
    - Attributes

### โครงสร้าง
  <html>
      <head>
          <ส่วนหัวของเว็บหรือการ link ข้อมูลต่างๆมาใช้ในหน้าเว็บ>
          <ที่สำหรับใส่ internal css>
      </head>

      <body>
          <ส่วนเนื้อหาของหน้าเว็บ>
          <ที่สำหรับใส่ internal js <script tag>>
      </body>

  </html>
